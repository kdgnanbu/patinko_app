# patinko_app

## 概要
パチンコ風の挙動を再現したWebアプリケーションです。  
JavaScriptによるロジック実装を目的として制作しました。

## 公開URL  
https://kdgnanbu.github.io/patinko_app/

---

## 使用技術
- HTML
- CSS
- JavaScript

---

## アプリ内容・機能
- ユーザーの入力値から期待値を計算する
- 重み乱数を使った確率の計算
- JavaScriptによる画面描画・更新処理

※ フレームワークは使用せず、基本的なJavaScriptのみで実装しています

---

## 工夫した点
- シンプルなデザインで使いやすいようにしました
- JavaScriptのみで実装できるようにしました
- 重み乱数を使うことでパチンコ特有の確率の計算ができるようにしました

---

## 苦労した点
- 重み乱数を使った計算式
- それぞれのボタンで違う処理
